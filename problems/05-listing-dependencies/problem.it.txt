npm non è solo per l'installazione di roba. Ti mostra anche cosa hai 
già installato (le tue dipendenze).

Puoi farlo usando il comando `npm ls`.

Esegui questo comando nella tua directory di lavoro. Se ci dovesserp essere
eventuali problemi npm ti avviserà restituendo un messaggio "!ERR".

Se tutto ti sembra apposto, esegui `$ADVENTURE_COMMAND verify OK`, o
`$ADVENTURE_COMMAND verify NOT OK` se npm segnala un problema".
